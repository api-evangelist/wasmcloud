# wasmCloud (wasmcloud)
wasmCloud is a CNCF incubating platform for building distributed applications with WebAssembly components. It provides a runtime connecting portable Wasm actors to infrastructure capabilities through declarative linking.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/wasmcloud/refs/heads/main/apis.yml)

## APIs

### wasmCloud Control Interface API
NATS-based API for managing the wasmCloud lattice including starting/stopping actors and providers, establishing links, querying inventories, and managing deployments.

**Human URL:** [https://wasmcloud.com/docs/](https://wasmcloud.com/docs/)

#### Tags:
 - Control API, NATS, Lattice Management

#### Properties
- [Documentation](https://wasmcloud.com/docs/)

### wasmCloud Application Deployment Manager (wadm) API
Declarative application deployment API using YAML manifests for components, providers, and links with self-healing reconciliation across the lattice.

**Human URL:** [https://wasmcloud.com/docs/ecosystem/wadm/](https://wasmcloud.com/docs/ecosystem/wadm/)
